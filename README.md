# Mental Health & Self-Discovery App - Support Website

This is the technical support website for the Mental Health & Self-Discovery iOS app.

## Setup GitHub Pages

1. Push this repository to GitHub
2. Go to your repository settings
3. Navigate to "Pages" section
4. Under "Source", select "Deploy from a branch"
5. Select the branch (usually `master` or `main`) and the `/docs` folder
6. Click "Save"
7. Your website will be available at: `https://[your-username].github.io/[repository-name]/`

## Local Development

To preview the website locally, simply open `docs/index.html` in your web browser.

## Customization

Before deploying, please update the following in `index.html`:

- Replace `apple277315540@163.com` with your actual support email address if needed
- Update the copyright year if needed
- Add your actual app store link if desired
